# 小克 - 私人聊天助手

一个本地运行的 AI 聊天助手，具备对话记忆和自动学习能力。

## 快速开始

### 第一步：安装 Python

去 [python.org](https://python.org) 下载 Python 3.10+，安装时**勾选 "Add to PATH"**。

### 第二步：获取 API Key

1. 打开 [platform.deepseek.com](https://platform.deepseek.com) 注册账号
2. 在「API Keys」页面创建一个 Key
3. 充值 **10 元**就能用很久（日常聊天一个月几块钱）

### 第三步：填写 Key

用记事本打开 `app.py`，找到这一行：

```python
api_key="sk-your-deepseek-api-key",
```

把 `sk-your-deepseek-api-key` 替换成你的真实 Key。

### 第四步：安装依赖

在项目目录打开终端（在文件夹地址栏输入 `cmd` 回车），运行：

```bash
pip install -r requirements.txt
```

### 第五步：启动！

```bash
python app.py
```

浏览器打开 **http://127.0.0.1:5000**，开始和你的小克聊天！

## 功能说明

| 功能 | 怎么用 |
|---|---|
| 💬 聊天 | 输入框打字，回车发送 |
| 🧠 记忆 | 自动搜索相关历史，AI 回复时参考 |
| 🧠 学习 | 点「学习」按钮，AI 总结你的偏好 |
| 📋 查看记忆 | 点「记忆」按钮，看小克记住了什么 |
| 🗑 清空记忆 | 点「清空」按钮，重新开始 |
| 📝 教知识 | 在聊天里发文章/链接，小克记住要点 |

---

## 自定义性格

编辑 `config.json`，改完重启即可：

```json
{
  "name": "小克",          // 名字，随便改
  "tone": "温柔幽默",       // 说话语气
  "style": "回答简洁",       // 回复风格
  "rules": [                // 自定义规则，想加几条加几条
    "永远用中文回复",
    "叫我老板"
  ]
}
```
